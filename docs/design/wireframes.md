# NAMC Website Wireframes

## Overview
Low-fidelity wireframes for the NAMC website covering public pages, member portal, and admin interfaces. These wireframes focus on layout, user flow, and functionality rather than visual design.

## Wireframe Legend
- **โก** Content blocks/sections
- **โข** Interactive elements (buttons, links)
- **โฃ** Forms/input fields
- **โค** Images/media placeholders
- **โฅ** Navigation elements
- **โฆ** Cards/containers
